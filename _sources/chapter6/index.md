# Summary

(glossary)=
## Glossary

| | |
| :--- | :--- |
| **Node** | A point in the network representing a person |
| **Edge** | A line connecting two nodes, representing a relationship |
| **Degree** | Number of connections a node has |
| **Undirected** | Edge without direction (A-B, not A?B) |
| **Broker** | Person connecting otherwise separate groups |
| **Cluster** | Group of densely connected nodes |
| **Ego network** | A node and all its direct connections |
